Domande frequenti

Le risposte alle domande più comuni su ED Tools.

Cos’è ED Tools?

ED Tools è una raccolta di strumenti online semplici, veloci e gratuiti. È un progetto personale creato per aiutare chiunque abbia bisogno di calcoli rapidi, conversioni o utility utili nella vita quotidiana.

Chi ha creato ED Tools?

Il sito è stato creato da Emanuele, uno sviluppatore web appassionato di tecnologia, design e programmazione. Puoi saperne di più nella pagina Chi sono.

I tool sono gratuiti?

Sì, tutti gli strumenti presenti su ED Tools sono completamente gratuiti e accessibili senza registrazione.

Come vengono calcolati i risultati?

Ogni tool utilizza funzioni JavaScript semplici e trasparenti. Non vengono effettuati invii di dati a server esterni.

Posso suggerire un nuovo tool?

Certo! Puoi contattarmi tramite la pagina Contatti per proporre nuove idee o miglioramenti.

Il sito raccoglie dati personali?

No, ED Tools non raccoglie dati personali. È presente solo Google Analytics per analizzare il traffico in forma anonima.
